# Virtual Manipulative Math Berbasis RME - Belajar Bangun Datar Jadi Lebih Menyenangkan!

Media pembelajaran interaktif matematika SD berbasis *Realistic Mathematics Education* (RME) tentang Bangun Datar.

## Tech Stack
- React 19 + Vite 8
- TailwindCSS v4
- Framer Motion
- React Router

## Cara Menjalankan
```bash
npm install
npm run dev
```
